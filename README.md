# 🚀 ShopPilot AI

AI-powered retail business intelligence platform that transforms sales CSVs into actionable insights, forecasts, executive summaries and downloadable reports.

---

## ✨ Features

- 📊 Interactive Dashboard
- 📈 Sales Forecasting
- 🤖 AI Business Insights
- 📦 Inventory Alerts
- 📄 PDF Report Generation
- 🎯 Executive Decision Center
- 📉 Business Health Score
- ⚡ Accelerated Analytics Dashboard

---

## 🏗 Architecture

```
CSV Upload
      │
      ▼
 FastAPI Backend
      │
 ├── KPI Engine
 ├── Forecast Engine
 ├── Business Score
 ├── Decision Center
 ├── AI Insights
 └── PDF Generator
      │
      ▼
 React Dashboard
```

## 🚀 Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
