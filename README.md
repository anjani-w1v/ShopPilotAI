# 🚀 ShopPilot AI

Transform raw retail sales data into actionable business insights using AI-powered analytics, forecasting, executive dashboards and decision support.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/anjani-w1v/ShopPilotAI
```

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
